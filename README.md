# COMP4300 Project 1 – Course Registration System

This project is an ASP.NET MVC web application that allows users to manage courses, students, and course registrations.

## Features
- Create, edit, delete courses
- Create, edit, delete students
- Register students for courses
- View registered courses
- Cancel registrations

## Technologies
- ASP.NET Core MVC
- Entity Framework Core (InMemory Database)
- xUnit (Unit Testing)

## How to Run
1. Open the solution in Visual Studio
2. Run the project (F5)
3. The app will launch at https://localhost:xxxx

## Unit Tests
Includes automated tests for:
- CourseController
- RegistrationController
